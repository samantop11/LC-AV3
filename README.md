# TABUADA EM C
uma tabuada criada em c le o numero digitado pelo usuario e o multiplica do 1 ao 10 usando while para repedicao.
## funcionalidade
- le o numero digitado mpelo usuario.
- imprime a tabuada.
- usa while para criar repeticoes.
